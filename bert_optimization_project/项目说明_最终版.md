## 📋 项目概况

本项目**完全自实现GEMM kernel**（不调用cuBLAS），并实现多个融合算子，对BERT模型进行深度优化。

### 核心成果

| 指标 | 结果 |
|------|------|
| **GEMM性能** | 达到cuBLAS的76% ⭐ |
| **整体加速** | 平均1.53×（提升34%） |
| **最高加速** | 1.70×（batch 64，提升41%） |
| **GEMM精度** | 误差1.83×10⁻⁴ |

## 📊 技术亮点

### 1. GEMM自实现（3个版本）

| 版本 | 性能 | 关键技术 |
|------|------|---------|
| 基础Tiled | 10% cuBLAS | Shared Memory Tiling |
| Register Tiling | 54% cuBLAS | 每线程8×8块 |
| **BERT特化768** | **76% cuBLAS** | **维度特化+循环展开** ⭐ |

### 2. 融合算子（4个）

| 算子 | 提升 | 关键技术 |
|------|------|---------|
| LayerNorm+Residual | 45% | Welford算法 |
| Fast GELU | 38% | tanh近似 |
| Online Softmax | 23% | 单遍扫描 |
| GEMM+Bias+GELU | 融合 | 三算子融合 |
